# YOLO Agricultural Detection Visualizer

**中文名称：** YOLO农业检测可视化系统

**项目全称：** 红星农场玉米出芽监测系统（农科院项目）

一个基于Web的YOLO目标检测结果可视化工具，专门用于农业场景中的农作物检测、统计和出芽率计算。本项目为农科院委托开发，应用于黑龙江省北安市红星农场（隶属北大荒集团北安分公司，耕地面积超41万亩）的玉米出芽监测场景。

## 项目背景

### 核心问题

- 🌾 **玉米出芽窗口短**：人工巡田难以及时发现"缺苗带"
- 📏 **耕地面积大**：41万亩耕地，传统人工巡田效率低、成本高
- ⏰ **时间紧迫**：需要在72小时黄金补苗期内快速定位缺苗区域

### 解决方案

- 🚁 利用无人机航拍 + YOLO检测技术
- 🔲 通过可旋转/可平移网格判定每格是否含苗
- 🎯 未变色区域即为补苗优先区，辅助农场主精准决策

## 功能特性

- 📷 **YOLO数据解析**：支持加载航拍图片和YOLO标注文件（.txt格式），自动解析检测结果
- 🔲 **自适应网格系统**：可自定义网格大小、旋转角度和平移偏移，适配不规则田垄
- 🔄 **实时可视化**：网格热力视图实时生成，含苗格高亮显示，空白格标记缺苗风险
- 📊 **智能统计**：自动统计每个网格中的检测对象数量，支持网格聚合分析
- 🌱 **出芽率计算**：自动统计检测苗数，输入播种总量即可秒级计算出芽率百分比
- 🖱️ **交互式操作**：支持鼠标缩放、平移、点击标记网格，提供流畅的用户体验
- ✏️ **人工复核机制**：支持人工标红或调整计数，形成"算法检测+人工复核"闭环
- 🎨 **自定义样式**：可自定义检测框和网格颜色，适配不同使用场景

## 使用方法

1. 选择图片文件（支持常见图片格式）
2. 选择对应的YOLO标注文件（.txt格式）
3. 调整网格大小、颜色等参数
4. 查看统计结果和出芽率

## 技术栈

### 前端技术
- **HTML5/CSS3**：暗色主题面板，现代化UI设计
- **Fabric.js 5.3.0**：Canvas图形渲染库，支持高性能图形操作
- **原生JavaScript**：纯前端实现，无需后端支持

### AI检测技术
- **YOLOv5/YOLOv8**：支持YOLO格式的检测结果解析
- **YOLO txt格式解析**：支持类别、中心点坐标、宽高、置信度等字段

### 样本数据
- `FTA03328.JPG`：航拍图像样本
- `FTA03328_detections.txt`：YOLO检测标签样本

## 核心技术实现

### 1. YOLO检测结果解析
- 解析YOLO txt格式（类别、中心点坐标、宽高、置信度）
- 将检测框坐标转换为画布坐标系统
- 按网格单元聚合苗点，统计每个网格的检测数量

### 2. 自适应网格系统
- **网格大小可调**：适配不同行距（10-2000像素）
- **网格旋转角可调**：适配田垄倾斜（-90°至90°）
- **网格平移偏移可调**：精确定位（±500像素）
- **实时刷新机制**：与画布操作同步，误差控制在5像素内

### 3. 热力视图生成
- 使用Fabric.js批量绘制高亮矩形
- 含苗格高亮显示，空白格标记缺苗风险区域
- 网格热力视图实时更新，支持缩放和平移

### 4. 人工标注功能
- `manualCellStates`数据结构管理人工标注状态
- 画布点击事件支持人工标红标记
- 支持调整计数，修正算法误检
- 算法检测与人工复核相结合，提升准确性

### 5. 出芽率计算系统
- 自动统计检测到的苗数
- 支持输入播种总量（正整数）
- 秒级计算出芽率百分比
- 实时刷新出芽率看板

## 操作提示
